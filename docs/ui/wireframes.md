# Logical wireframes (App + Web)

## A) Pantallas clave (App móvil)

### 1. Home / Búsqueda
- **Header**: logo + ciudad de salida (Irlanda) + icono de perfil (opcional)
- **Filtro rápido**: botones 200€ / 300€ / 400€
- **Selector de fecha**: calendario simple (mes + día)
- **CTA**: "Buscar paquetes"
- **Sección sugeridos**: 3–5 tarjetas destacadas

### 2. Lista de paquetes
- **Filtros activos** (chips): precio, fecha, destino (si aplica)
- **Tarjeta paquete**:
  - Imagen
  - Título (Destino + duración)
  - Precio
  - Fecha salida
  - Botón "Ver detalle"

### 3. Detalle de paquete
- **Hero**: imagen + título
- **Resumen**: precio, fecha, duración, ciudad de salida
- **Incluye**: vuelo, hotel, actividades (lista)
- **No incluye** (opcional)
- **CTA**: "Reservar interés"
- **Guardar**: icono de favorito

### 4. Reservar interés
- Formulario: nombre, email, teléfono (opcional)
- Checkbox: "Acepto recibir información"
- Botón: "Enviar"
- Confirmación simple

### 5. Favoritos (opcional)
- Lista de tarjetas guardadas
- Eliminar de favoritos

## A) Pantallas clave (Web)

### 1. Home
- Hero con buscador (precio + fecha)
- Destinos populares
- CTA "Ver paquetes"

### 2. Listado
- Panel lateral de filtros (precio, fecha, destino)
- Resultados en cards

### 3. Detalle
- Layout en dos columnas: info + CTA reserva
- Bloque "Incluye" / "No incluye"

### 4. Reserva
- Formulario igual que app
- Mensaje de confirmación
