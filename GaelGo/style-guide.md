# GaelGo — UI Style Guide (v0.1)

## Brand keywords
Clean · Budget‑first · Trustworthy · Simple

## Color palette (Temple Bar inspired)
- **Primary (Gold):** #D4AF37
- **Primary Dark:** #A9872A
- **Accent (Burgundy):** #7B1E3A
- **Background:** #0F0F10
- **Surface:** #1A1B1E
- **Border:** #2A2D33
- **Text Primary:** #F5F3EE
- **Text Secondary:** #C9C2B8
- **Success:** #2E8B57
- **Warning:** #C7782A

## Typography
- **Font:** Inter (fallback: Arial, sans‑serif)
- **H1:** 28px / Bold
- **H2:** 22px / Semi‑bold
- **Body:** 14–16px / Regular
- **Caption:** 12px / Regular

## Spacing & Radius
- **Base spacing:** 8px (8/16/24/32)
- **Card radius:** 12px
- **Button radius:** 10px
- **Input radius:** 8px

## Components

### Buttons
- **Primary:** bg #1F6FEB, text #FFFFFF, height 44px
- **Secondary:** bg #E6EEF9, text #1F6FEB, border #C7D7F5

### Cards
- **Background:** #FFFFFF
- **Border:** 1px solid #E2E8F0
- **Shadow:** 0 6px 20px rgba(17,24,39,0.08)
- **Padding:** 16px

### Chips (budget filters)
- **Default:** bg #E6EEF9, text #1F6FEB
- **Active:** bg #1F6FEB, text #FFFFFF

### Inputs
- **Background:** #FFFFFF
- **Border:** 1px solid #D1D5DB
- **Focus:** border #1F6FEB, shadow 0 0 0 3px rgba(31,111,235,0.2)

### App header
- **Background:** #F0F4FA
- **Title color:** #111827

## Example card layout
- Image (top, 16:9)
- Title
- Price + date row
- Includes (small tags)
- CTA “View details”
